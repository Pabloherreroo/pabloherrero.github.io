## ðŸ›  To Run Locally
If you wish to clone and customize this repository, you will need to have **Jekyll** installed.

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/pabloherreroo/pabloherreroo.github.io.git](https://github.com/pabloherreroo/pabloherreroo.github.io.git)
    cd pabloherreroo.github.io
    ```
2.  **Install Dependencies (if you use Bundler):**
    ```bash
    bundle install
    ```
3.  **Serve the Site:**
    ```bash
    bundle exec jekyll serve
    ```
The site will be accessible locally at `http://127.0.0.1:4000/`.
